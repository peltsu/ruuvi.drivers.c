enum ri_timer_mode_t

Timer abstraction. Allows creating single-shot and repeated timers which call a function at interval. Timer IDs must be defined with PLATFORM_TIMER_ID_DEF(timer_id) The timer will schedule the functions to run, so there might be a lot of jitter in timeout.
